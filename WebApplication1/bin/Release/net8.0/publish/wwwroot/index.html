API Endpoints

Use the generated token in the X-API-Token header to access these endpoints:

GET /api/InventoryApi/inventory
Get inventory information
GET /api/InventoryApi/items
Get all items in the inventory
GET /api/InventoryApi/aggregated
Get aggregated statistics (price stats, popular words, etc.)
GET /api/InventoryApi/discussions
Get discussions for the inventory
GET /api/InventoryApi/summary
Get complete summary with statistics
